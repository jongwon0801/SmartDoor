#### 디바이스 등록 순서 (콘솔 기준)

✅ [1] AWS IoT Core → “사물(Thing)” 등록

```less
1. AWS 콘솔 → IoT Core 접속

2. 좌측 메뉴에서 "사물" (Things) → “사물 생성”

3. “단일 사물 생성” 선택 (라즈베리파이 1대 기준)

4. 사물 이름 입력 (예: wikibox_pi_01)

5. 디바이스 유형 선택 안 해도 됨 (옵션)
```

✅ [2] 인증서 생성 및 연결
```less
1. “새 인증서 자동 생성” 선택

2. 생성된 다음 항목 모두 다운로드:

3. 인증서 (certificate.pem.crt)

4. 개인 키 (private.pem.key)

5. 루트 CA (AmazonRootCA1.pem)

이 인증서를 라즈베리파이로 복사해놓기
```

✅ [3] 정책 만들기 및 연결

새 정책 생성 : AllowAll

```less
{
  "Version": "2012-10-17",
  "Statement": [
    {
      "Effect": "Allow",
      "Action": [
        "iot:Connect",
        "iot:Publish",
        "iot:Receive",
        "iot:Subscribe"
      ],
      "Resource": "*"
    }
  ]
}

```


✅ [4] AWS IoT Core 엔드포인트 확인
```less
IoT Core → 좌측 하단 “설정”

“데이터 끝점”에서 MQTT 엔드포인트 복사

인증서 파일 다운로드

🔒 certificate.pem.crt
🔑 private.pem.key
🔐 AmazonRootCA1.pem
```
