#### 자동화(Automation)
```less
automation:
  - alias: "Input 1 On Turn On Relay 1" # 자동화의 이름
    trigger:
      - platform: state
        entity_id: binary_sensor.입력_센서_1 # 위에서 등록한 입력 센서의 entity_id
        to: "on" # 입력 센서의 상태가 '켜짐'으로 변경될 때
    action:
      - service: switch.turn_on
        entity_id: switch.릴레이_1 # 릴레이 1의 entity_id

```

# 기본 명령어 형식

#### 에드온 모니터링
```less
1. 홈어시스턴트 웹 인터페이스에서 왼쪽 사이드바의 "설정" 메뉴를 클릭하세요.

2. 설정 페이지에서 "애드온" 항목을 선택합니다.

3. 오른쪽 하단의 "애드온 스토어" 버튼을 클릭합니다.

4. 검색창에 "Node-RED"를 입력하고 검색하세요.

5. 검색 결과에서 "Node-RED" 애드온을 찾아 클릭합니다.

6. "설치" 버튼을 클릭하여 애드온을 설치합니다.

7. 설치가 완료되면 "시작" 버튼을 눌러 애드온을 실행합니다.

8. "웹 UI 열기" 버튼을 클릭하면 Node-RED 인터페이스에 접속할 수 있습니다.

이렇게 설치하시면 Node-RED를 통해 Modbus 통신을 시각적으로 모니터링하실 수 있습니다
```
