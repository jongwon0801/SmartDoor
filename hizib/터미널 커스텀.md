#### 파일별 색상 적용하려면 GNU ls 설치
```
brew install coreutils
```

#### ~/.zshrc 파일을 열어서 아래 내용을 추가
```
export LS_COLORS="di=34:ex=32:ln=36:*.py=35:*.sh=33:*.json=31:*.zip=36"
alias ls='gls --color=auto'
```

#### 설정내용
```
di=34 → 디렉터리(폴더) = 파란색

ex=32 → 실행 파일 = 연두색

ln=36 → 심볼릭 링크 = 하늘색

*.py=35 → 파이썬 파일 = 자주색

*.sh=33 → 쉘 스크립트 파일 = 노란색

*.json=31 → JSON 파일 = 빨간색

*.zip=36 → ZIP 파일 = 하늘색
```

#### 적용 & 확인
```
source ~/.zshrc

ls -la
```
